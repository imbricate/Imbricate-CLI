## Full Documentation

Visit [imbricate.io](https://imbricate.io/).

### Using Yarn

Yarn 1.x

```sh
yarn global add imbricate
```

Yarn 2.x

```sh
yarn dlx -p imbricate
```

### Using NPM

```sh
npm install -g imbricate
```

## Usage

```sh
imbricate --help
```
